## <u>en-video-app</u>

##### <概要>

英語のドキュメンタリービデオを集めたサイトです。
動画サイトを作ってみたいと思い作成しました。
字幕が動画の再生に合わせて切り替わったり、字幕部分にちょっとしたメモができる機能をつけています。
URL: https://en-video-app.vercel.app/
